# 연산자
프로그래밍 언어에서 특정 연산을 하도록 하는 문자

## 산술 연산자
사칙연산과 같은 작업을 하는 연산자
- +, -, *, /
- ++a, --a, a++, a--

## 대입 연산자
특정 값에 연산을 한 값을 바로 적용할 때 사용하는 연산자
- a += 1, a -= 1, a *= 1, a /= 1

## 논리 연산자
boolean타입을 위한 연산자
- !(NOT), &&(AND), ||(OR)
- 연산 순서
  - NOT > AND > OR

## 비교 연산자
두 값을 비교할 때 사용하는 연산자
- \>, <, <=, >=, ==, !=, ===, !==
- ===, !== vs ==, !=
  - ===과 !==는 두 값의 타입까지 검사
  - ==과 !=는 두 값이 일치하는지 확인(ex - 0, false를 동일하게 인식)
