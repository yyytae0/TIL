### git repo 최초 등록

1. git init
2. git remote add origin url
3. git add .
4. git commit -m 'message'
5. git push origin master


### git repo 등록 이후

1. git pull
2. git add .
3. git commit -m 'message'
4. git push origin master

### 새로운 환경에서 git 활용

1. git copy url
2. git add .
3. git commit -m 'message'
4. git push origin master
5. 자격증명관리자에서 git 정보 제거
